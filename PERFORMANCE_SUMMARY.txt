HOW TO VERIFY (5 MINUTES)
═════════════════════════════════════════════════════════════════════════════

Method 1 — Chrome DevTools (Fastest)
──────────────────────────────────────
1. Open http://localhost:8000
2. Press F12 → Lighthouse tab
3. Click "Analyze page load"
4. Read Performance score (target >90)

Method 2 — Cache Headers
──────────────────────────────────────
curl -I http://localhost:8000/assets/site.css
→ Should show: Cache-Control: public, max-age=31536000, immutable

Method 3 — Slow Network Test
──────────────────────────────────────
DevTools → Network → Throttling: "Fast 3G" → Hard reload (Ctrl+Shift+R)
→ LCP should be <2.5s, CLS <0.1

# Body‑Family Copy Guide  
*Voice & Messaging for ZELEX Concierge Atlas*

---

## 1. Brand Voice Principles  

| Principle | What it means on the site | Example phrasing |
|----------|---------------------------|-----------------|
| **Quiet‑Luxury** | Language is calm, refined, and never shouted. | “A whisper of curve, a promise of comfort.” |
| **Editorial Concierge** | Copy reads like a high‑end magazine feature that guides a personal shopper, not a product catalog. | “Your next silhouette, curated by experts.” |
| **Measurement‑Led** | Every claim is rooted in the WHR/BWR ranges that define each family. No vague “big” or “small” claims. | “WHR 0.68–0.72 delivers the timeless hourglass.” |
| **Intent‑First Journey** | The user flow moves from **intent → silhouette family → body architecture → character → private inquiry**, and copy mirrors that progression. | “Start with what you envision, we’ll reveal the perfect family.” |
| **Elevated Tone, Not Hype** | Avoid over‑the‑top superlatives; instead, offer nuanced, aspirational language. | “Subtle power, measured grace.” |

> **Voice tip:** Write as if you’re a personal stylist whispering options to a discerning client. Every sentence should feel like a privileged recommendation, not a sales pitch.

## 3. Global CTA Strategy  

| Funnel Stage | Primary CTA | Copy Tone | Escalation Logic |
|--------------|------------|-----------|------------------|
| **1️⃣ Discover / Quiz** | *“Begin Your Silhouette Journey”* | Inviting, exploratory, quiet‑luxury. Uses “discover” language that aligns with first‑time intent. | Leads user to a **family recommendation** based on measured preferences (WHR/BWR). |
| **2️⃣ Compare** | *“Side‑by‑Side Comparison”* | Analytical, editorial, still personal. Highlights measurement ranges (e.g., “WHR 0.68–0.72 vs 0.60–0.65”). | After family is identified, user chooses models to compare, deepening engagement. |
| **3️⃣ Inquire / Concierge** | *“Request a Private Consultation”* | Exclusive, concierge‑style, final‑step. Reinforces that only a **private inquiry** unlocks pricing, bespoke finishing, and delivery. | The copy now references the selected family’s premium (e.g., “Your Icon selection, +30 % premium, awaits a private briefing”). |

**Copy escalation pattern:**  
- **Quiet curiosity** → **Measured insight** → **Exclusive access**  
- Each step includes a subtle reminder of the *measurement‑led* nature (“Your chosen WHR falls within the Icon family range”).  
- The final CTA always frames the inquiry as a **personal concierge conversation**, never as a generic “contact us”.

## 4. Do’s & Don’ts  

| Do | Don’t |
|----|-------|
| Use the exact WHR & BWR ranges when describing a family. | Say “big” or “small” without referencing the numeric ranges. |
| Write in a calm, editorial tone that feels like a private magazine feature. | Use hype‑filled superlatives such as “the best ever” or “unbelievable”. |
| Reference the **premium %** for each family when positioning price. | Invent new premium figures or discount language. |
| Guide the buyer step‑by‑step (intent → family → body → character → inquiry). | Present a flat product grid or catalogue view. |
| Keep CTA language concierge‑focused (“private consultation”, “personal briefing”). | Use generic CTAs like “Buy Now” or “Add to Cart”. |
| Align buyer archetype language with the family description (e.g., “Photographer” for Icon). | Mismatch archetype (e.g., “Fitness guru” for Siren). |
| Keep all copy measurement‑centric, referencing WHR/BWR where relevant. | Make vague statements about “shape” without measurement support. |

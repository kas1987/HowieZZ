2. Red/amber/green status by family lane.

4. Next-week commitments by team lead.

Required inputs each week:

- Product: body-status update.
- Growth: family funnel metrics.
- Creative: asset readiness map.
- CX: objections and conversion blockers.

## 6) Escalation Triggers

Escalate to Howie immediately if any occurs:

- Classic commission slips by more than 2 weeks.
- Quiz dead-end reappears in any live route.
- Icon or Muse inquiry share drops >20% week-over-week without explained cause.
- Sculpt capex request appears before pilot validation criteria are met.

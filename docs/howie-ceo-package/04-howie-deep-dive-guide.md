## Question-to-Analysis Map

Read first:

What to extract:

- Launch hierarchy logic (primary, secondary, in-development, suppressed)
- Family penetration by market
- ROI tradeoffs by family

Decision output:

- Confirm or modify current staged launch model

## 30-Minute Leadership Deep-Dive Format

Use this structure in review meetings:

1. Question definition (3 minutes)
2. Evidence recap from one deep brief (10 minutes)
3. Risk and tradeoff discussion (10 minutes)
4. Decision and owner assignment (7 minutes)

Record each decision in `05-howie-decision-log-template.md` immediately after the session.

## Decision Hygiene Rules

1. If a decision changes launch hierarchy, update the two-page brief the same day.
2. If a decision changes product gating, update both this guide and the team execution guide.
3. If a decision depends on new data, append source links in the dedicated brief library.

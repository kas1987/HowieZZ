# PDR-FE-001: Homepage Concierge Landing

## Objective
Redesign `index.html` from a character atlas landing page into a premium buyer-guidance landing page.

## Scope
Claude Design should create a visual/UX patch for:
- improved hero hierarchy
- trust strip
- three buyer paths: Find Your Match, Compare Bodies, Browse Characters
- six-family chooser preview
- body compare preview module
- concierge closing CTA
- mobile-safe CTA hierarchy
- preservation of existing series rail and featured character sections

## Files allowed
- `index.html`
- `assets/site.css`
- optionally `assets/site.js` only for nav/CTA linkage

## Files blocked
- `db/*.json`
- build scripts
- image/video assets

## Acceptance criteria
- First-time buyer understands primary action within 10 seconds.
- Compare Bodies appears above or near the first major decision block.
- Six-family taxonomy is visible.
- Classic and Sculpt are represented honestly as in-development when applicable.
- Existing data-driven sections still render.
- Page works on mobile and desktop.

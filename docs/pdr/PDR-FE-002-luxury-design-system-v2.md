# PDR-FE-002: Luxury Design System v2 Visual Refinement

## Objective
Refine `assets/site.css` into a coherent premium design system.

## Scope
Claude Design should define/refine:
- surface hierarchy
- button variants
- card depth
- family tile styling
- status chips
- verified/estimated/concept/shoot-pending states
- form panels
- comparison table design
- mobile spacing rhythm
- accessible focus states

## Acceptance criteria
- Shared classes are reusable across homepage, cards, compare, quiz, contact, and character pages.
- Design avoids excessive gold/glow.
- Focus states are visible.
- Reduced-motion and missing-image states remain safe.

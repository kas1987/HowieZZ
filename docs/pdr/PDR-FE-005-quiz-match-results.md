# PDR-FE-005: Quiz Match Results Funnel

## Objective
Improve `quiz.html` result UX so the quiz becomes a real buyer-routing funnel.

## Scope
Claude Design should improve:
- ranked top 3 family results
- match percentage visual treatment
- explanation panel: why this family matched
- recommended character cards
- compare recommended bodies CTA
- browse family CTA
- ask concierge CTA
- in-development family handling

## Acceptance criteria
- User sees more than one result.
- Winning family has clear reasoning.
- Result has a next step into compare/contact.
- In-development families do not dead-end the user.

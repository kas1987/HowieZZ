# PDR-FE-006: Concierge Intake UX

## Objective
Upgrade `contact.html` into a premium concierge intake flow.

## Scope
Claude Design should create/refine:
- context summary card for selected character/body/quiz/compare
- better buyer-intent fields
- trust sidebar
- privacy/18+ copy
- made-to-order expectations
- no-pressure consultation language
- success/error states

## Acceptance criteria
- 18+ consent remains required.
- Form feels consultative, not generic ecommerce.
- No PII is used in analytics/logging.
- Mailto fallback remains available unless backend endpoint is configured.

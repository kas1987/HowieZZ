# PDR-FE-008: Mobile Navigation and CTA Pass

## Objective
Make the mobile journey feel premium and conversion-safe.

## Scope
Claude Design should review/refine:
- sticky nav behavior
- mobile menu spacing
- CTA hierarchy
- tap target size
- hero mobile crop/spacing
- card/grid stacking
- compare table mobile behavior
- contact form mobile layout
- footer trust signals

## Acceptance criteria
- Primary action is always obvious on mobile.
- No text feels cramped.
- Tap targets are safe.
- Compare and contact remain usable on phone.

# PDR-002: Homepage Concierge Landing

## Objective
Rebuild `index.html` into a buyer-intent landing page that routes users into match quiz, body comparison, character browsing, and concierge inquiry.

## Current gap
Current homepage still follows the original Atlas pattern: hero, three doors, series rail, featured characters. It needs stronger buyer confidence and decision-flow modules.

## Scope
- Add trust strip: made-to-order, measured silhouettes, private inquiry, 18+ only.
- Add primary CTAs: Find Your Match, Compare Bodies, Browse Characters.
- Add six-family chooser preview.
- Add body-compare preview block.
- Add concierge closing CTA.
- Preserve series rail and featured characters as supporting modules.

## Acceptance criteria
- A first-time buyer understands what to do within 10 seconds.
- Compare Bodies is visible above and below the fold.
- Six family tiles display correctly on desktop and mobile.
- No dependency on unavailable images for core usability.
- Page remains static-host compatible.

## Dependencies
PDR-001, PDR-003.

## Owner
Cartographer.
